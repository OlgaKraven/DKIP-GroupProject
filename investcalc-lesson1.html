1. Цели занятия

Команда должна:

  1. Выбрать предметную область — конкретного «клиента» из отрасли.
  2. Определить бизнес-проблему — зачем клиенту нужен расчёт окупаемости ИС.
  3. Собрать исходные данные — CAPEX, OPEX, обучение, период анализа, эффекты.
  4. Подготовить модель расчётов по формулам ЛР6:
    • TCO
    • ROI
    • Payback Period
    • чувствительность ±20%
  5. Создать черновую модель расчётов в Python или C#.

2. Организация командной работы и GitHub

2.1. Создание командного репозитория

Каждая команда создаёт репозиторий с шаблонной структурой:

investcalc-teamXX/
│
├── docs/                # описание предметной области, модели, отчёты
├── src/                 # серверная часть (Python/C#/Node)
│   ├── api/             # будущие контроллеры/эндпоинты
│   ├── services/        # расчётные модули
│   └── models/          # модели данных
│
├── tests/               # тестовые данные / unit tests
├── data/                # исходные данные проекта
├── docker/              # dockerfile, docker-compose
└── README.md            # описание проекта

17. Разработка модели расчёта (по ЛР6)

17.1. TCO

Локальный сценарий:

TCO_local = CAPEX + (OPEX * period) + training

Облачный сценарий:

TCO_cloud = (subscription * period) + training

17.2. ROI

ROI = ((TotalBenefits - TCO) / TCO) * 100

Где:

TotalBenefits = (savings + revenue_growth) * period

17.3. Payback Period

PP = TCO / (savings + revenue_growth)

17.4. Анализ чувствительности ±20%

Изменяются:

  • CAPEX
  • OPEX
  • subscription
  • экономия
  • рост доходов

Расчёт:

value_low  = value * 0.8
value_high = value * 1.2

18. Черновая модель (Python и C#)

18.1. Вариант Python

def calc_local(capex, opex, training, period, savings, revenue):
    tco = capex + opex * period + training
    benefits = (savings + revenue) * period
    roi = (benefits - tco) / tco * 100
    pp = tco / (savings + revenue)
    return {"tco": tco, "roi": roi, "payback": pp}

18.2. Вариант C#

public static class InvestCalc
{
    public static (double TCO, double ROI, double Payback) CalcLocal(
        double capex, double opex, double training,
        int period, double savings, double revenue)
    {
        var tco = capex + opex * period + training;
        var benefits = (savings + revenue) * period;
        var roi = ((benefits - tco) / tco) * 100;
        var pp = tco / (savings + revenue);
        return (tco, roi, pp);
    }
}

Черновой расчёт сохраняется в:

  • src/model/local_calc.py или
  • src/services/CalcService.cs.
